# Installation
Run setup.sh to configure dotfiles.

# Requirements:
* i3 Windows Manager
* Urxvt terminal
* ZSH shell
* Sublime Text 3
* rofi

Sublime Text 3 plugins:

* Color Highlight
* Compare Side-by-Side
* Local History
* Monokai - Spacegray
* Pretty JSON
* SideBarEnhancements
* Theme - Spacegray
