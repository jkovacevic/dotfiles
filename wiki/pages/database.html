Database

Create localhost database

sudo apt-get install postgresql postgresql-client postgresql-contrib libpq-dev

sudo -u postgres psql postgres

When in psql shell dotcomma is important otherwise command won't work!

alter user postgres with password 'postgres';

create database database_name;

\connect database_name;

create schema eu;

Backup database with pg_dump

/usr/bin/pg_dump --format=c --create --dbname=dbname --username=username --host=hostname --port=5432 --file=/tmp/db.pg_dump

Restore database with pg_restore

/usr/bin/pg_restore --format=c --clean --dbname=dbname --username=username --host=localhost --port=5432 /tmp/db.pg_dump
